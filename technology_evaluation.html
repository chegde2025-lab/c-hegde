ETL Processing Strategy

for chunk in read_table(chunksize=50000): transformed = transform(chunk) write_to_database(transformed)
  • Chunk-based processing
  • Push joins to database
  • Efficient SQL queries

IDMC → AWS Batch Migration Strategy

  1. Inventory all IDMC workflows
  2. Document mappings and transformations
  3. Convert transformations into Python ETL scripts
  4. Package scripts into Docker containers
  5. Deploy containers to AWS Batch
  6. Run validation alongside existing pipelines
  7. Cut over production workloads
